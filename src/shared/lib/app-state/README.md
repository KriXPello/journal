# app-state

Эта библиотека содержит только глобальное клиентское состояние приложения, которое не привязано к конкретной странице:

- кэш загруженных данных;
- глобальный loading state.

Сюда нельзя добавлять:

- persistence-логику;
- вычисления предметной области;
- локальное состояние конкретных страниц.
