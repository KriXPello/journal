# app-data

Эта библиотека отвечает за формат резервных копий и операции с backup-файлами:

- versioned backup schema;
- сериализация и парсинг backup-файлов;
- генерация имен backup-файлов.

Сюда нельзя добавлять:

- прямую работу с IndexedDB;
- UI настроек;
- произвольные доменные модели вне контракта backup.
